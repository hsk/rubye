# RubyE

## RubyEとは?

RubyE はRubyのような式言語です。
RubyE 自体ではS式やXMLのように意味を持ちません。
RubyE はシンプルなツリーデータを表すデータフォーマットです。

## RubyE 例

- 1.後置演算子
    ```
    a++
    ```
- 2.前置演算子
    ```
    !a
    ```
- 3.左結合２項演算子
    ```
    a+b+c
    a else b
    a elseif b
    ```
    else や elseif もまた２項演算子です。
- 4.右結合２項演算子
    ```
    a = b = c
    ```
- 5.括弧
    ```
    (a)
    [a]
    {a}
    if a b c end
    do a b c end
    ```
    ifやdo等も括弧です。
- 6.括弧演算子
    ```
    a(b)
    a[b]
    a{b}
    ```
    括弧演算子は、左に結合します。

とりあえず、こんな感じの複数種類の式を持ちます。
